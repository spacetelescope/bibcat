textdata.py

This module is a class purely meant to be inherited by the various *Classifier classes and inherits Base() class. In short, the ClassifierBase class is a collection of methods used by different classifier types.

The primary methods and use cases of ClassifierBase include: * classify_text: Base classification method, overwritten by various classifier types during inheritance.
